# Android ADB & Scrcpy Universal Toolkit

This repository contains a complete universal set of tools, scripts, and guides for Android device management using:

- ADB (Android Debug Bridge)
- Scrcpy (screen mirroring & remote control)
- Wireless debugging
- VPN debugging
- SystemUI controls
- Root utilities
- Device automation

This toolkit works with:
- Samsung tablets / phones
- Pixel tablets / phones
- Any rooted or non-rooted device

---

## Structure

### adb/
ADB commands, scripts, and troubleshooting.

### scrcpy/
Scrcpy setup, wireless mode, VPN mode, and automation.

### tools/
Platform tools, drivers, and system configurations.

---

## Purpose

This repo provides:
- Universal scripts that work on any device
- Clean documentation
- Tested ADB/Scrcpy configurations
- VPN remote connection instructions
- Lockdown utilities for kiosk setups
- Tools for IT deployments or personal automation projects
